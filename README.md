# _Numbers to Words_

#### _c# Exercise for Epicodus, 2016_

#### By _**Ben Logue and Callan McNulty**_

## Description

Given a number in numerals as input, will output the number in words.

## Setup/Installation Requirements

* _clone from github_
* _run dnu restore_
* _run dnx kestrel_
* _play_
